# IoT 配置界面

基于 Vue 3 + Element Plus 的物联网设备配置管理系统，支持星闪（SLE）和 WiFi 双模式无线网络配置。

## 项目简介

本项目是一个现代化的 IoT 设备配置界面，提供直观的 Web 界面用于配置和管理物联网设备的网络参数、接口设置和系统配置。

### 主要功能

- **无线模式配置**
  - 支持 SLE（星闪）和 WiFi 双模式切换
  - 主机模式、从机模式、主从机模式三种工作模式
  
- **星闪网络配置**
  - 主机参数设置（名称、MAC、扫描从机、接入限制）
  - 从机参数设置（名称、MAC、自动广播、心跳包）
  - 主从机混合配置
  - 扫描从机设备功能
  
- **WiFi 网络配置**
  - AP 模式（热点）配置
  - STA 模式（客户端）配置
  - AP+STA 混合模式配置
  - WiFi 热点扫描功能
  - SOCKET 网络协议配置
  - 客户端接入限制（白名单/黑名单）
  
- **LAN 配置**
  - IP 地址、子网掩码、网关配置
  - DHCP 服务器设置
  
- **接口管理**
  - 以太网口开关控制
  
- **系统设置**
  - 时区配置
  - NTP 服务器设置

## 技术栈

- **前端框架**: Vue 3 (Composition API)
- **UI 组件库**: Element Plus
- **状态管理**: Pinia
- **路由**: Vue Router
- **构建工具**: Vite
- **开发语言**: JavaScript

## 项目结构

```
iot-config-ui/
├── src/
│   ├── components/          # 公共组件
│   ├── layouts/            # 布局组件
│   ├── views/              # 页面组件
│   │   ├── network/        # 网络配置页面
│   │   ├── interface/      # 接口配置页面
│   │   └── system/         # 系统配置页面
│   ├── stores/             # Pinia 状态管理
│   ├── router/             # 路由配置
│   ├── menu/               # 菜单配置
│   └── main.js             # 入口文件
├── public/                 # 静态资源
├── dist/                   # 构建输出
└── package.json            # 项目配置
```

## 快速开始

### 开发模式
